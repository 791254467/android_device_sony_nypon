android_device_sony_nypon
=========================

Unoficial CM-11.0 for Sony Xperia P

Getting Started :

    mkdir cm-11.0
    cd cm-11.0
    repo init -u git://github.com/Legacy-Xperia/manifest.git -b cm-11.0
    repo sync -j16

You are ready to build :

    . build/envsetup.sh
    brunch cm_nypon-userdebug
